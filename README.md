# Python Notepad Application

## Overview
This is a simple Notepad application built using Python. The project was created to practice GUI development, file handling, and event-driven programming concepts.

## Features
- Create new text files
- Open existing text files
- Edit and save content
- Basic user-friendly interface

## Technologies Used
- Python
- Tkinter (GUI Library)
